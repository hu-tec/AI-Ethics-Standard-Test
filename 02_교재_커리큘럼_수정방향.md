# AI 윤리 교재·커리큘럼 수정 방향 MD

> 목적: 지금까지 만든 교재와 커리큘럼을 앞으로 어떻게 수정해야 하는지 정리한 문서입니다.

---

## 1. 현재 교재·커리큘럼에서 수정해야 할 점

### 1.1 수정 필요성

기존 방향은 AI 윤리의 개념, 글로벌 기준, 국가별 정책을 설명하는 데 강점이 있었다.  
하지만 현재 사업 방향은 **AI 윤리 자격·진단·교육·시험·인증·컨설팅 사업**이므로 교재도 아래처럼 바뀌어야 한다.

| 기존 방향 | 수정 방향 |
|---|---|
| AI 윤리 설명 중심 | 진단·교육·시험·자격·인증 연계형 |
| 성인·직장인 중심 | 유치원~초중고~대학생~직장인~전문가~심사관까지 대상 확대 |
| 글로벌 정책 소개 중심 | 국내외 사례 + 공식자료 탐색 실습 + 진단표 작성 |
| 윤리 원칙 중심 | ESG·보안·신뢰·컴플라이언스 중심 |
| 강의자료형 | 시험문제·실습과제·보고서·체크리스트 포함 교재형 |
| 단일 난이도 | 급수별·대상별 난이도 체계 |

## 2. 교재 전체 구조 제안

| 8 | AI 윤리 진단 실습 | 무료/일반/전문가/최고전문가 진단 구조 | 자기진단 및 해석 |
| 9 | AI 윤리 시험 대비 | OX, 객관식, 사례형, 보고서형 | 모의시험 |
| 10 | AI 윤리 인증과 컨설팅 | 기업 진단, 개선계획, 증빙, 인증 | 기업 윤리진단 보고서 |

## 3. 대상별 커리큘럼

### 3.1 유치원·초등 저학년

| 목표 | 내용 | 방식 |
|---|---|---|
| AI 안전 습관 | AI가 알려준 말을 모두 믿지 않기 | 그림 카드 |
| 개인정보 보호 | 이름, 학교, 전화번호, 사진을 함부로 넣지 않기 | 상황극 |
| 착한 AI 사용 | 친구를 놀리는 이미지·음성 만들지 않기 | OX 게임 |
| 보호자 연결 | 자녀 AI 안전진단으로 연결 | 부모 안내문 |

### 3.2 초등 고학년·중학생

| 목표 | 내용 | 방식 |
|---|---|---|
| AI 결과 검증 | AI 답변이 틀릴 수 있음을 이해 | 검색 비교 |
| 저작권 기초 | AI 그림·글도 남의 권리를 침해할 수 있음 | 사례 퀴즈 |
| 딥페이크 경계 | 가짜 영상·음성 구분과 신고 | 영상 사례 토론 |
| AI 과제 윤리 | 숙제·리포트에 AI 사용 시 표시 | 사용일지 작성 |

### 3.3 고등학생·대학생

| 목표 | 내용 | 방식 |
|---|---|---|
| 학업 윤리 | 리포트, 논문, 발표자료 AI 활용 기준 | AI 사용 선언문 |
| 진로·취업 윤리 | 자기소개서, 포트폴리오, 면접 AI 활용 | 사례형 문제 |
| 글로벌 기준 | EU, NIST, UNESCO 등 공식자료 이해 | 사이트 탐색 보고서 |
| 자격 연계 | 일반 3급~2급 과정 | 모의시험 |

### 3.4 직장인

| 목표 | 내용 | 방식 |
|---|---|---|
| 업무 보안 | 회사자료, 고객정보, 내부기밀 입력 금지 | 업무 시나리오 |
| 생성물 검수 | 보고서, 광고, 번역, 이미지 검토 | 검수표 작성 |
| 책임과 승인 | AI 결과 사용 책임, 승인 절차 | 부서별 규칙 작성 |
| B2B 진단 | 직원 AI 윤리 이해도 측정 | 조직 분석표 |

### 3.5 전문가·강사·컨설턴트

|---|---|---|

| 컨설팅 | 기업 리스크 분석과 개선안 | 컨설팅 보고서 |

## 4. 자격 급수별 교재 방향

| 자격 | 교재 방향 |
|---|---|
| 유치원·초등 9~7급 | AI 안전습관, 개인정보, 친구를 해치지 않는 AI 사용 |
| 초등·중등 6~4급 | AI 결과 검증, 저작권, 딥페이크, 과제 윤리 |
| 중고등 3~1급 | 글로벌 기준 기초, 생성형 AI 활용 책임, 사례 판단 |
| 일반 3급 | AI 윤리 입문, 개인·업무 활용 기본 |
| 일반 2급 | 직무별 AI 사용, 보안, 저작권, 표시, 검수 |
| 일반 1급 | 조직 내 AI 사용수칙, 리스크 관리, 교육 가능 수준 |
| 전문 2급 | 진단표 작성, 사례 교육, 컨설팅 보조 |
| 전문 1급 | 기업 진단, 교육 설계, 리스크 보고서 작성 |
| 최고전문가/심사관 | 인증심사, 법·규제·ESG·컴플라이언스 심화 |

## 5. 실습 설계

## 6. 교재가 사업으로 연결되는 구조

```text
교육생이 AI 윤리 필요성을 이해
→ 무료 진단
→ 취약 영역 확인
→ 유료 교육 추천
→ 급수 시험 응시
→ 자격 취득
→ 전문가 과정 진입
→ 기업 진단/인증/컨설팅 참여
```

## 7. 결론: 수정해야 할 핵심

교재와 커리큘럼은 반드시 다음 5가지를 반영해야 한다.

1. **대상별 분화**: 유치원부터 최고전문가까지.
2. **자격 급수 연계**: 1~9급, 일반 1~3급, 전문 1~2급.
3. **진단 테스트 연계**: 교육 전후 진단으로 성장 확인.
4. **컴플라이언스 강조**: 윤리는 자율이 아니라 앞으로 제도화될 기준.
5. **사업 전환 구조**: 교육 → 시험 → 자격 → 인증 → 컨설팅 → 전문가 인프라.
