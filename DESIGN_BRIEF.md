### 📄 Page 2 — 시험 접수 신청서 (mockup-registration)

**URL 패턴:** `/exam/register`

#### DB 연결 테이블
`exam_registrations`, `exams`, `exam_types`, `users`, `coupons`, `user_points`, `user_agreements`, `payments`

#### 레이아웃 구성

**[헤더]** — 메인과 동일  

**[페이지 제목 + 4단계 진행 표시]**  
```
✓ 시험 선택  ──  [2. 신청 정보 입력]  ──  3. 결제  ──  4. 접수 완료
```
- 완료 단계: 초록 원 + 체크마크
- 현재 단계: 초록 채워진 원 + 흰 숫자, 굵은 텍스트
- 미완료 단계: 흰 원 + 회색 테두리 + 회색 숫자

**[본문 2컬럼: 좌측 2/3 + 우측 1/3]**

**좌측 — 신청서 폼 (위→아래 섹션별 카드)**

① **시험 종목 선택** 카드  
- 3개 카드 (가로 배치): AI 윤리⚖️ / AI 번역🌐 / AI 프롬프트💬  
- 선택된 카드: 초록 테두리 + 연초록 배경  
- 미오픈 카드(AI 프롬프트): opacity 낮게  

② **등급 선택** 카드  
- 2개 카드: 기초(₩35,000) / 전문(₩60,000)  
- 등급별 설명 (시험 대상/문항/시간/합격 기준)  

③ **시험 일정 선택** 카드  
- 3개 일정 카드 (가로 배치):
  - 2026.07.12 (토) 10:00 / 잔여 128석 ← 선택됨 (초록 테두리)
  - 2026.07.19 (토) 14:00 / 잔여 243석
  - 2026.08.02 (토) 10:00 / 잔여 300석

④ **개인 정보 확인** 카드  
- 회색 배경 안내: "가입 정보로 자동 입력됩니다"
- 2컬럼 폼 필드: 이름 / 생년월일 / 이메일 / 휴대폰  
- 증명사진 업로드 영역 (점선 테두리 드래그앤드롭 스타일)  
- 알림 수신 방법 체크박스 (카카오톡/이메일/문자)  

⑤ **약관 동의** 카드  
- "전체 동의" 큰 초록 배경 라벨
- 들여쓰기로 개별 항목 4개 (필수 3개, 선택 1개)
- 각 항목 우측에 "보기" 텍스트 링크

**우측 — 접수 요약 (sticky 고정)**

- **접수 요약 카드**: 시험종목/등급/일시/환경/성적발표 정보 테이블
- **결제 금액 카드**:
  - 응시료 ₩35,000
  - 쿠폰 코드 입력 + 적용 버튼 (인라인)
  - 포인트 사용 (보유 12,500P + 입력 필드)
  - 최종 결제액 강조 (₩35,000, 초록 큰 숫자)
  - 결제 수단 라디오 버튼 (카드/카카오페이/네이버페이/포인트 전액)
- **자격증 발급 카드**: 옵션 체크박스 + 추가 금액 ₩3,000
- **"결제하고 접수 완료 →" 버튼** (초록, 전체 너비, 큰 버튼)
- **환불 규정 안내** (노란 배경 경고 박스)

  - 중앙: 이름 + 취득 자격명 + 등급
  - 하단: 취득일 + QR코드
- 버튼: PDF 다운로드 / 링크드인 공유

**[탭4: 포인트 내역]**  
상단: 보유 포인트 총액 강조 표시  
테이블: 날짜 / 구분(적립/사용/만료) / 내용 / 변동량 / 잔액
